# epayco-frontend-challenge

CRUD de posts construido con Next.js 14 (App Router), TypeScript, Tailwind CSS,
React Query, Zustand y React Hook Form + Yup.

## Requisitos

- Node.js 18+ (recomendado)
- npm

## Instalación

```bash
npm install
```

## Variables de entorno

Copia el archivo de ejemplo:

```bash
cp .env.example .env
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Abre http://localhost:3000 en tu navegador.

## Build y producción

```bash
npm run build
npm start
```
